== Description ==
Analyse your theme and plugin code on this site and provide you a report of compatibility issues

Note
This plugin works only on WordPress sites hosted on one.com domain.

Support
Contact our support

== Installation ==
Automated Installation
To do an automatic install, log in to your WordPress dashboard, navigate to the Plugins menu, click Add New, search for \"one.com Under Construction\", and click to install.

Manual Alternatives
The manual installation method involves downloading our plugin and uploading it to your webserver via your favorite FTP application. The WordPress codex contains instructions on how to do this here.
